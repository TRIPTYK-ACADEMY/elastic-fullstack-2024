# BOILERPLATE FULLSTACK NODEJS

## Installation

### With Git

```
git clone https://github.com/TRIPTYK-ACADEMY/fullstack-tournai-boilerplate
```

```
rm -rf .git
git init
```

### With pnpx

```
pnpx degit https://github.com/TRIPTYK-ACADEMY/fullstack-tournai-boilerplate
```

Install packages:

```bash
pnpm i
```

## Scripts

### Dev environment

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Prod

```bash
pnpm start
```

### Lint

```bash
    pnpm lint
```
